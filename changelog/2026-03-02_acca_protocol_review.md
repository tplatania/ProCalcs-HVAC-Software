# 2026-03-02 — ACCA Protocol Review & DOE Engine Intel

## What
Received and analyzed ACCA's official MJ8 Software Review Protocol (54-page PDF).
Discovered ACCA is working with DOE on a common calculation engine (2028 launch).

## Key Findings

### ACCA Licensing Costs
- $10,000 non-refundable review fee
- $15,000 annual licensing fee (5% annual increase)
- 5-year term, auto-renews
- Total 5-year cost: ~$83,000

### DOE Common Calculation Engine
- ACCA working with DOE to build shared Manual J calc engine
- Software companies plug in their own front-end
- Expected launch: 2028
- Software companies still welcome to submit own engines for review

### New ACCA Requirements (not yet in formal docs)
- Must output ACCA's Plan Review Form
- Must output composite J1 Form + Worksheet A
- Must provide open license for ACCA staff use

### Competitive Math (Wrightsoft charges $500-$1,500/year)
- At $600/year ProCalcs pricing: need 25 subscribers just to cover ACCA fee
- First 25 customers generate zero profit
- Could run at loss for 12-18 months on licensing alone

## Strategic Decision
- Validator FIRST (no ACCA approval needed, pure margin)
- Build experience layer (AI visualization, conversational reviewer, smart data collection)
- Wait for DOE common engine (2028) before pursuing ACCA approval
- Let Validator revenue fund everything
