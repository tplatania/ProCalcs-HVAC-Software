# ProCalcs HVAC Design Software

**Proprietary HVAC load calculation software built in 4 phases.**

From Load Calc Validator to ACCA-Approved Platform.

## Phases
1. **Validator** (Months 1-4) — Upload & verify Manual J calcs. Revenue from day 1.
2. **Full Engine** (Months 5-12) — Complete Manual J 8th Edition calculation engine.
3. **ACCA Approval** (Months 12-18) — Mandatory certification for permit-ready docs.
4. **SaaS Launch** (Month 18+) — Sell to 105,000+ HVAC contractors nationwide.
